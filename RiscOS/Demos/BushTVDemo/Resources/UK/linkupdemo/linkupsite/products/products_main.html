Overview

LinkUp Systems products support the ARM architecture. There are two types of products:

  • Integrated single chip solutions with embedded ARM CPU and DSP
  • Support chips for Intel StrongARM processor

In order to aid the system OEM in the system development effort, LinkUp also provides complete system solutions. The system solution consists of:

  • Development System
  • System software (Windows CE and Symbian OS software drivers and OAL)
  • Applications Program Interface
  • Manufacturing kits
