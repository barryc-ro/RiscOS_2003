The Advanced Palette Window.




This window allows greater control over the palette. The values that define the greater control here are derived from other values in the simplified window. This help page describes the differences between the two windows.

This window also more closely replicates a command line application that preceeded this, as such the order of which items appear is in a different order than the simplified window.

Saturation

  • Error Diffuse Density; This is the position of maximum ink. It is used to space the colour map to within a range from 0 to this value.

  • Brightness This has two values these are for error diffusion and half-tone methods of colour approximation. Half-tone has been removed for the simplified window as it is not used in the majority of cases.

  • Contrast This exactly the same.


Colour Biases and Maximum Levels

    This is again the same as the simplified window.


Grey Level Replacement

    The difference here is the second value. This value is used as the maximum level, in most cases this is 100%. For this reason this value is left out from the simplified dialogue.
