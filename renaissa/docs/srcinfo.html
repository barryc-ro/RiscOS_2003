Overview

srcinfo is a utility which allows two files of RISC OS Sprite or template file format to be compared under unix. This allows a simple comparison to be made to see which file has items added/removed/modified therefore which is the most up-to-date during a merge process.

Usage

srcinfo [-sprite|-template] <filename1> <filename2>
flag description
-sprite process the two files as sprite file format
-template process the two files as template file format

When invoked srcinfo will load each of the two files specified and process them as sprite/template files (note: auto filetype detect is not implemented). It will then generate a md5 hash of each individual sprite/window within the file and use this to detect differences between the two files. The following results may occur :

caseoutput generated
identical spritesnothing
sprite/window missing from one filewarning given
sprite/window differentwarning given
