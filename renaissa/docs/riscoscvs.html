The command line CVS program has been ported to RISC OS. The latest released version of the port is available on Julian Smith's shared disc as Share::JSmith.$.!Jules.ANCWork.!CVS.

All documentation is in the TechWriter file !CVS.!Help. You must read this before using RISC OS CVS.

Note that the above link will only work if you are using RISC OS and have both ShareFS and !FilerURLs running.
